# 🧬 Bioinformatics Study Notes

Personal study resources and notes for **MS in Bioinformatics** at Northeastern University.

---

## 📖 About

This repository contains my learning materials, notes, and resources as I progress through my graduate studies in Bioinformatics. It serves as both a personal reference and a way to track my learning journey.

## 🎯 Areas of Focus

### Transcriptomics & RNA-seq
- Bulk RNA-seq analysis pipelines
- Single-cell RNA sequencing (scRNA-seq)
- Differential expression analysis
- Quality control and preprocessing

### Computational Tools
- **Alignment:** STAR, HISAT2, Kallisto
- **Quantification:** featureCounts, Salmon
- **Analysis:** DESeq2, Seurat, Scanpy
- **Workflow:** Nextflow, Snakemake

### Programming Languages
- Python (pandas, NumPy, scikit-learn)
- R (tidyverse, ggplot2, Bioconductor)
- Bash scripting
- SQL basics

### Statistical Methods
- Differential expression statistics
- Multiple testing correction
- Dimensionality reduction (PCA, UMAP, t-SNE)
- Clustering algorithms
